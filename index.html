Top Exam Tips for Unit 4
1
FDE Cycle : Always name the specific bus at each step — address bus, data bus, control bus. Mention the Program Counter and the instruction register. These are the mark-scoring words.
2
Cloud Storage : This is often a 6-mark question. Give 3 advantages AND 3 disadvantages, each with a developed explanation of WHY it matters. Don't just say "accessible anywhere" — say why that's useful.
3
Truth Tables : Always add intermediate columns for complex expressions. Show your working — you earn marks for correct intermediate columns even if the final output has an error.
4
OS Functions : For "state four functions of an OS" — pick from: hardware management, file management, scheduling, paging/virtual memory, user interface, authentication. One per mark — don't write paragraphs.
5
Compiler vs Interpreter : The three key differences: whole program vs line by line · object code produced vs not · faster at runtime vs slower. Always explain WHY — e.g. "slower because the interpreter translates the same code every time the program runs."
6
Storage Comparison : Magnetic = spinning platters + seek time. Optical = laser + pits/lands. Solid-state = NAND flash + electron pools. Defragmentation is ONLY for HDDs — SSDs need no defragmenting.

Topic 4.1 — Machines & Computational Models
Understand the Input–Process–Output (IPO) model and how all computers follow it
Describe the sequential computational model and its characteristics
Explain the parallel computational model and what it requires (OS + software)
Describe the multi-agent computational model: autonomous agents, negotiation, coordination
Distinguish between parallel and multi-agent models (one task split vs separate independent tasks)
Topic 4.2 — Hardware
Understand the von Neumann stored-program architecture
Describe the function of CPU components: CU, ALU, registers, clock, buses
Explain the fetch–decode–execute cycle in detail (naming specific buses and registers)
Compare RAM, ROM, cache and virtual memory (volatile, non-volatile, speed, purpose)
Explain firmware and the boot process (POST, BIOS/UEFI, loading the OS)
Describe factors affecting CPU performance: clock speed, cores, cache size
Explain how data is stored on magnetic, optical and solid-state secondary storage
Discuss cloud storage advantages and disadvantages (⭐ 6-mark topic)
Understand embedded systems and the Internet of Things (IoT)
Topic 4.3 — Logic
Construct and interpret truth tables for AND, OR and NOT
Build truth tables for combined expressions using intermediate columns
Apply operator precedence correctly: brackets → NOT → AND → OR
Write logic statements (IF...THEN conditions) for given real-world scenarios
Topic 4.4 — Software
Distinguish between application software, system software and utility software
State and explain the key functions of an operating system
Explain scheduling (CPU time-slicing) and paging (virtual memory)
Compare GUI and CLI interfaces (advantages and disadvantages of each)
Describe utility software: antivirus, spyware, firewall, defragmenter, compression, backup
Understand computer simulation and modelling: advantages, limitations, examples
Topic 4.5 — Programming Languages
Explain machine code, assembly language and the role of mnemonics
Understand the role of an assembler (1:1 mnemonic to machine code)
Describe high-level languages and why they need translators
Compare compilers and interpreters: translation method, output, speed, error handling, distribution
Explain when to use a compiler (distribution) vs interpreter (development/debugging)
